# RFC 003

## Контекст

Для домашки 003 нужно создать класс для хранения неотрицательных чисел в BCD-формате. Для хранения байт следует использовать Uint8Array, где в одном байте нужно хранить одну цифру (со звездочкой (\*) в одном байте 2 цифры)

```
// Абстрактный класс для примера
abstract class BCD {
    // Можно усложнить и сделать поддержку number | bigint
    constructor(num: number) { /* ... */ }

    abstract toBigint(): bigint;
    abstract toNumber(): bigint;
    abstract toString(): string;

    // Возвращает значение разряда BCD числа на указанной позиции.
    // Отрицательная позиция означает разряд "с конца".
    abstract at(index: number): number;
}

const n = new BCD(65536n);

console.log(n.toBigint()); // 415030n
console.log(n.toNumber()); // 65536

console.log(n.at(0)); // 6
console.log(n.at(1)); // 5

console.log(n.at(-1)); // 6
console.log(n.at(-2)); // 3
```

## Решение

Я сделаю свой класс MyBCD, который будет принимать число в конструкторе и под капотом конверирует его в Uint8Array, чтобы было удобно хранить 1 число = 1 байт. Я реализую методы toBigint, toNumber, toString и at для доступа к цифрам по индексам.

Конвертация в Uint8Array будет работать так: я пройду по каждому разряду числа в цикле по типу while (число > 0) { цифра = число % 10; сохранить цифру в массив интов; обновить число = Math.floor(число / 10); }. По сути откидываю последнюю цифру через деление на 10 и выбрасывание дробной части. В итоге я обхожу все разряды в числе с конца.

toNumber буду делать так - сделаю редьюс массива интов, буду умножать каждую цифру на 10 в степени ее позиции, потом складывать с накопленным результатом.

toString будет просто конкатенировать все цифры в строку по тому же принципу что и toNumber, только конкатенировать надо будет в обратном порядке. Сделаю разделение блоков по 4 цифры через нижнее подчеркивание для удобства чтения.

toBigint будет работать аналогично toNumber, только вместо умножения на 10 в степени позиции, я буду использовать BigInt(10) \*\* BigInt(позиция).

at будет просто возвращать цифру из массива интов по индексу, учитывая что отрицательные индексы означают позицию с конца. Поэтому для отрицательных индексов я буду делать индекс = array.length + index, чтобы получить правильный индекс в массиве. Если индекс выходит за границы массива, то возвращаю 0
