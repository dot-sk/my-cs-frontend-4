# 004 Про операции циклического сдвига влево и вправо

## Контекст

Циклический сдвиг — это операция, при которой элементы последовательности (массива, строки, битового регистра) перемещаются так, что элементы, покидающие один конец, немедленно появляются на другом конце. Последовательность замыкается в кольцо

```ts
// Биты отбрасываемые из-за переполнения слева дополняются справа
cyclicLeftShift(0b10000000_00000000_00000000_00000001, 1) ===
  0b00000000_00000000_00000000_00000011;

// Биты отбрасываемые из-за переполнения cправа дополняются слева
cyclicRightShift(0b10000000_00000000_00000000_00000001, 2) ===
  0b01100000_00000000_00000000_00000000;
```

### Частные случаи

- Сдвиг на 0 позиций - Последовательность не изменяется.
- Сдвиг на половину длины - Сдвиг влево на n/2 и сдвиг вправо на n/2 дают одинаковый результат (при чётном n)
